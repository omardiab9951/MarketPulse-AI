## 📦 Dataset

The dataset file `electronics_full_cleaned.7z` is available in the [`Dataset`](https://github.com/omardiab9951/MarketPulse-AI/tree/Dataset) branch of this repository.

### How to Download & Extract

1. Go to the [`Dataset`](https://github.com/omardiab9951/MarketPulse-AI/tree/Dataset) branch
2. Click on `electronics_full_cleaned.7z`
3. Click **Download**
4. Extract the file using [7-Zip](https://7-zip.org):
   - **Windows:** Right-click the file → 7-Zip → Extract Here
   - **CMD:** 
```cmd
     "C:\Program Files\7-Zip\7z.exe" e electronics_full_cleaned.7z
```
   - **Linux/Mac:**
```bash
     7z e electronics_full_cleaned.7z
```
5. You will get `electronics_full_cleaned.csv` ready to use
